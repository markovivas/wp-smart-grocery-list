# WP Smart Grocery List

**Versão:** 1.0.0
**Requer:** WordPress 5.0 ou superior
**Licença:** GPLv2 ou posterior

O WP Smart Grocery List é um plugin WordPress desenvolvido para facilitar a criação e o gerenciamento de listas de compras de forma inteligente.

## Descrição

Este plugin permite organizar suas compras diretamente no painel do WordPress. Com uma interface limpa e intuitiva, você pode criar novas listas e gerenciar itens rapidamente. O plugin foi desenvolvido com foco em performance e facilidade de uso.

## Funcionalidades Principais

*   **Lista de Compras Inteligente:** Organize seus itens de forma centralizada.
*   **Gerenciamento de Listas:** Crie uma "Nova Lista" facilmente.
*   **Pronto para Tradução:** Inclui arquivo `.pot` para tradução (Português, Inglês, etc.).

## Instalação

1.  Faça o upload da pasta `wp-smart-grocery-list` para o diretório `/wp-content/plugins/` do seu servidor.
2.  Ative o plugin através do menu **Plugins** no WordPress.
3.  Comece a usar acessando a nova área administrativa do plugin.

## Como Usar

1.  Navegue até a seção do plugin no menu do WordPress.
2.  Clique no botão **"Nova Lista"** para iniciar.
3.  Adicione os itens desejados à sua lista.
